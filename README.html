A real-time digital piano built on the MSPM0G3507 microcontroller using C and ARM Cortex-M0+ assembly. The system integrates a 5-bit DAC and 4-bit keyboard to generate musical tones with precise timing.
Features
Real-time audio generation via SysTick interrupts
Low-latency keypress-to-sound mapping (<10ms)
Support for multiple musical notes and smooth playback
Demonstrates embedded systems programming, hardware-software integration, and signal processing fundamentals
